# Scale Degree & Chord Extension Flashcards

A browser-based practice app for naming chord extensions and scale degrees relative to a root. Answer on an on-screen piano keyboard, with optional sound, light/dark theme, and configurable roots and extensions.

## Features

- **Practice** — work at your own pace with flip cards and immediate feedback.
- **60s Challenge** — timed mode for quick drills.
- **Settings** — choose which root notes and extensions (3rds, 5ths, 7ths, 9s, 11s, 13s, etc.) to include.
- **Results** — review accuracy and mistakes after a session.

## Run locally

This project is static HTML, CSS, and JavaScript. Serve the folder with any static file server so assets load correctly.

### Option A: Node (`serve`)

```bash
npx --yes serve .
```

Open the URL shown in the terminal (often [http://localhost:3000](http://localhost:3000)).

### Option B: Python

```bash
python -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080).
